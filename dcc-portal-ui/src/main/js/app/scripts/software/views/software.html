How to verify?

To verify the signatures you will need to ensure you have downloaded our public key and imported it:

From a key server:

$ gpg  --recv-key 0xAFE5D0B6

From a file:

$ gpg --import icgc-software.pub

Once the public key has been imported you can verify the signature:

$ gpg dcc-storage-client-0.0.27-dist.tar.gz.asc

If successful, the ouput should contain the following:

gpg: assuming signed data in `dcc-storage-client-0.0.27-dist.tar.gz'
gpg: Signature made Mon  5 Oct 12:38:09 2015 EDT using RSA key ID AFE5D0B6
gpg: Good signature from "ICGC DCC (Sign public software releases) "
