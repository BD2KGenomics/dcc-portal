ICGC and Amazon have partnered to offer low-cost and scalable computing solutions for analysing ICGC data on AWS EC2. The ICGC alignment and variant data on AWS S3 is managed by the ICGC Data Coordination Center (DCC) which is also developing software tools to search and access the repository content from AWS EC2 instances. Raw data and associated metadata produced by the ICGC PCAWG study as well as the one submitted by ICGC's members to the European Genome Archive (EGA) is periodically gathered, curated and incrementally uploaded to this repository for the benefit of the cancer research community. An up-to-date summary of the repository's data content is provided on the right. Our user guide provides step-by-step instructions on how to access and analyse ICGC data on AWS.

PCAWG Workflows

The PCAWG project has created standardized, whole genome workflows for alignment using BWA-Mem and variant calling using best practice processes from the Sanger Institute, the Broad Institute, and DKFZ / EMBL. These are made available to the research community via a "launcher" AMI that allows users to analyze their data using the same workflows. Currently supporting the BWA alignment workflow, this system makes it easy for users to load their data into AWS S3, launch many VMs simultaneously to analyze data in parallel, and write the resulting alignments back to S3. Over time, the full range of PanCancer workflows will be made available.
